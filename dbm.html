Dead by Moonlight

Overview

Dead by Moonlight is a game concept UI based on the famous game, dead by daylight published by Behaviour Interactive studio.

My inspiration for this project came after playing the game with my friends. The concept for this game is to kill other hunters which is comparatively different from the dead by daylight. The game is single player horror game. I created a UI for this survival horror video game and also designed the micro interactions.

Role

UI designer and Interaction designer

Timeline

November 2020

Dveloping the Designs and Results

I created my high fidelity mock-ups in Adobe XD and then created prototyping paths between the artboards. While navigating through the UI in Adobe XD, I recorded my screen and later imported the footage in Adobe Premier Pro to add overlays to make the UI feel alive.
